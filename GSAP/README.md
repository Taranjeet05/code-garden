
# GSAP Learning Journey

## 001 - Tweening Basics

In this folder, I practice the fundamentals of tweening and learned:
- `gsap.to()` - Animate to target values
- `gsap.from()` - Animate from specified values
- `gsap.fromTo()` - Animate from and to specific values

These methods are used for simple animations.

### 001-Project
A completed project demonstrating animation techniques using tweening concepts.
